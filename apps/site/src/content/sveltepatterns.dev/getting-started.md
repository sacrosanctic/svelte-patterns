# From React

See the syntatical differences on [Component Party](https://component-party.dev/?f=react-svelte5)

Mental model differences include:

- JSX is JS based, Svelte is HTML based
- hooks are just regular functions
- everything is fine-grained as explained by [Rich Harris' Rethinking reactivity](https://youtu.be/AdNJ3fydeao)
- one file, one component; Single File Component (SFC)
