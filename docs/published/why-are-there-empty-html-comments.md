title: What do these empty HTML comments do
---

## They are hydration markers

::: info Written for
Svelte `5.17.3`
:::

These are markers used at runtime for the purposes of [hydration](https://svelte.dev/docs/kit/glossary#Hydration).

```html
<!---->
<!--[-->
<!--]-->
<!---->
<!---->
```
